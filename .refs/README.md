# 参考项目目录

该目录只用于存放参考仓库、示例实现或接口资料，不作为当前主项目源码目录。

## 使用约定

- `refs/` 只放参考项目
- 每个参考项目单独放在自己的子目录中
- 目录命名尽量使用项目名或用途名，避免含糊名称
